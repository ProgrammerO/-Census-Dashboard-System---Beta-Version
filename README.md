# Census Dashboard System - Beta Version C++ 

This project is currently in beta and under active development.

**### #The full source code is private at this stage to protect the project while development is ongoing.**

## Current Features
- Patient CRUDE operations (Add, Delete, Find and update(asking which info wanna update ).
- validate data entry for the MRN in case invalid format like @#$$%%# is prohibited.
- File-based data storage
- Console-based user interface

## Beta Limitations
- Some session management features are still under development.
- No availability of  user based access in the system now (working on it :-)  )
- File-based storage is used temporarily.
- UI version is planned for future development.
- Sessions Total price has bug which shows 0  in the total price  whenever creating a new session  so which I didn't fix the bug yet.

- Creating a new menu called Reports to generate reports with specific columns or KPIs.
- C# desktop interface
- SQL database integration
- Role-based access
- Dashboard reports
